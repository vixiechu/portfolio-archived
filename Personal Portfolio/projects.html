PROJECTS

Web Technologies - iSongs

For one of my Level 4 assignments, I was required to create a working online store, similar to iTunes, where people could browse a library of available music, as well as having the option to register their own user account. I built this using PHP & MySQL, which we were the programming languages we were learning at the time. I found the languages initially tricky to master; but with applied practice I was able to successfully complete my assignment; obtaining a First as my overall grade. I didn't add any CSS to this; as I wanted to focus on the functionality and ensure the structure was correct. I will add a nice CSS layout in the future.
